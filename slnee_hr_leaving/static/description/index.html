Employee Leaving Request

In Case of some reason employee leave the job that time employee can put leaving request, HR officer and HR manager approved it.

Based on leaving request update employee contract.

Employee Leaving Request

Employee can put request for leaving with details of Leaving Reason, Leaving Request Date, Description and confirmed the request.



HR Department Validation

HR Officer check all the details also add Notice Start Date, Notice End Date, Exit Date, Contact Person and validated Leaving Request.



HR Department Approval

After validation, HR Manager will approve or refuse leave request.



Resignation Information Mail

HR Manager send resignation information to the employee with some details, Like Employee Notice Period, Exit Date, Contact Person.



Update Employee Form


After Approval of request by HR Manager update employee Leaving Date in employee form view.



Update Employee Contract


After Approval of request by HR Manager update employee contract.
