Operations Request

Apps used to sent the different operation request to the HR Department.

Configuration

HR Manager can add different types of request with related documents list.



Operation Request

Employee can request for operation to the HR Department.All related document will list out based on selected operations.



Request In progress

HR Officer will check operations request details and documents from employee, add person who's handle request and continue request in In Progress.



Operation Complete


Once operation will complete, HR officer will update record to Received.



Expense Generation


If any type of expense are occur during this operation then HR Officer can add and generate expense.
