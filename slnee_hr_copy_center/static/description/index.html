HR Copy Center Process

If any employee wants to generate hard copies of any official or personal document they can create a request. At the end of process expense will be generated as per employee & company contribution.

Expense will be paid by company & employee contribution will be directly deducted from employee's payslip.

Request For Document Hard Copy

Employee will create request, Attach soft copy & submit to Admin Manager.



Approve/Refuse Request

Admin Manager will add expense (company & employee contribution) details & approve request.



Set To In Progress

Admin Manager will set request in progress stage once actual process start.



Generate Expense

Admin Manager will set process on done stage once process will be done & generate expense.



Expense Process

Expense Manager will do expense approval process.



Employee Payslip Calculation

Current month expense (Employee contribution) will be deducted from employee's current month payslip.



Report

Excel report (can be filter date wise) with all details.
