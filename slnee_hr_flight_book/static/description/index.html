Employee Flight Booking

In case of Business tour/Meeting/Training employee will request to HR Manager for flight booking.

HR Manager will approve employee request & Admin Manager will book tickets for employee.

If booking is related to employee's personal work Admin Manager add employee contribution for expense. Which is directly deducted from employee's current month payslip (Salary).

Booking Request

Employee will create request & submit to HR Manager.



Approve/Refuse Request

HR Manager will approve/refuse request.



Set to Inprogress/Refuse Request

Admin Manager will set request to inprocess/refuse.



Ticket Received

Admin Manager will add expense (company contribution and employee contribution) & ticket details.



Generate Expense

Admin Manager will generate expense for booking.



Expense Process

Expense Manager will approve expense sheet.



Employee Payslip Calculation

Current month expense (Employee contribution) will be deducted from employee's current month payslip.



Admin Report

Excel report (can be filter date wise) with all details.
