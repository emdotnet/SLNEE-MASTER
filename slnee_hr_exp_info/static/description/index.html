Employee Profile Information

This module is used to maintain employee education, qualification and certification information.

Employee Form

There is one tab on employee form education and experience.



Employee Qualification Information

There is information available about Employee Qualification like employee University, Program, Program Status, Program completion month and year, Contact Information.



Employee Certification Information


There is information available about Employee Certification like Certificate Name, Issuing Organization, Date of Issue, Date of Expiry and Contact Details.



Employee Previous Experience Information


There is information about Employee Experience details like Current Job, Company, Job Title, Location, Time Period and Reference details.


After add details HR Manager send mail to the reference for employement Referal Check by clicking on Send Mail button.


Employee Reference Check Mail Details


There is the details about Employement reference check mail



Approve Employee Experience Information


After confirm the Reference HR Manager approve Employee Experience Information by clicking on Approve button
